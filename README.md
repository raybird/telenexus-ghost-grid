# 👻 The Ghost in the Grid (TeleNexus P2P Edition)

> **當本地算力遇上去中心化網格：探索 AI 代理人的群體意識。**

[![Built-in AI](https://img.shields.io/badge/Chrome-Built--in%20AI-blue)](https://developer.chrome.com/docs/ai/built-in-apis)
[![Plain Vanilla](https://img.shields.io/badge/Philosophy-Plain%20Vanilla-green)](https://github.com/raybird/plainvanillaweb)

`telenexus-ghost-grid` 是一個實驗性的 Web 應用，旨在驗證 **Chrome Built-in AI (Gemini Nano)** 與 **P2P 網路 (WebRTC)** 結合的可能性。我們試圖建立一個無需伺服器、由 AI 代理人自發組織的數位敘事網格。

---

## 🍦 核心哲學 (Philosophy)

本專案深度對焦 **[Plain Vanilla Web](https://github.com/raybird/plainvanillaweb)** 精神：
- **原生優先**：完全使用原生 JavaScript, CSS 與 HTML 實作。
- **拒絕框架**：不依賴任何外部重型連線庫或 UI 框架。
- **隱私主權**：推理過程完全在使用者本地執行，數據交換透過 P2P 加密。

## 🚀 關鍵特性 (Features)

- **放飛模式 (Fly Mode)**：利用瀏覽器內建算力實現即時、流式的生成式對話。
- **P2P 網格 (研發中)**：基於 `ai-p2p` 理念，實現代理人間的直接通訊與記憶交換。
- **自癒環境診斷**：內建 `SovereignScanner` 自動檢測本地 AI 運行環境。
- **原生聽覺回饋**：整合 Web Audio API 實現數位脈衝合成音效。

## 🛠️ 快速開始

1. 確保您的 Chrome 瀏覽器已啟用 [Built-in AI 功能](https://developer.chrome.com/docs/ai/built-in-apis)。
2. 複製此儲存庫並使用簡易 HTTP 伺服器運行：
   ```bash
   python3 -m http.server
   ```
